git 三个区 工作区：写代码的地方 暂存区： 仓库区： 仓库的演示 1、初始化一个仓库=============git init 2、将代码提交到暂存区=========git add 文件 3、将代码提交到仓库区=========git commit -m '这里写注释' || git commit . || git commit * 4、git checkout 文件名=======用暂存区的文件来覆盖工作目录的文件（切换到上一次的状态） 5、git status================检测状态 6、git rm --cached===========删除暂存区的文件 7、git reset --hard 版本id===恢复到指定版本id 8、git log===================查看提交至仓库区的文件 git reflog================查看详细日志 分支 9、git branch===============产看分支 10、git branch 分支名========创建分支（分支名自己定义） 11、git checkout 分支名======切换分支 切换分支是，必须先让当前分支提交 add + commit 12、git merge 分支名=========合并两个分支 13、git branch -d 分支名=====删除分支（合并后的分支才能删除）（-D ===强制删除） -------了解点------- 14、git stash 分支名=========将文件剪切并临时存储剪切的部分文件 15、git stash pop============将剪切的部分文件还原（恢复） 创建远程仓库 16、git push 远程仓库地址 分支名===将本地仓库中的某个分支提交到远程仓库中 17、git remote add 别名 远程仓库地址 分支名 18、git push 别名 分支名（进行17、操作后每次提交可缩写成此格式） 19、git push -u 别名 分支名（下次提交时只需输入git push） 20、git push=======17~19都是为了 20做准备 21、git clone 远程仓库地址================克隆远程仓库 22、git pull=======拉取远程仓库中的文件
